# [CAPFITOGEN](https://www.capfitogen.net/en/)

Scripts for [CAPFITOGEN](https://www.capfitogen.net/en/), corresponding to the zipped `scripts` folder provided for [CAPFITOGEN-3 local mode](https://www.capfitogen.net/en/access/capfitogen3-local-mode/).

This repository is an alternative source of downloading the scripts, and a base for developing the code further. 

How to cite: Parra-Quijano et al. 2021, https://repositorio.unal.edu.co/handle/unal/85787
